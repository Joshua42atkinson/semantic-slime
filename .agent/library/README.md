# Research Library

Welcome to the **Antigravity Research Library**.

This folder serves as the central repository for knowledge, best practices, and philosophical guidelines for our project.

## Contents

- **[Project Context](context.md)**: The single source of truth for the project vision, goals, and status. **Read this first.**
- **[Roblox Development](roblox_development.md)**: Technical guide on professional workflows using Rojo, Git, and VS Code.
- **[Gamification Theory](gamification_theory.md)**: Educational theory, Cognitive Load, and VaaM mechanics.
- **[Vibe Coding Manifesto](vibe_coding_manifesto.md)**: The "Antigravity" philosophy of flow-state development.
- **[Best Practices](best-practices.md)**: Specific coding rules, style guides, and "Do's and Don'ts".

## Usage

Use this library to:
1.  **Onboard**: Understand the "why" and "how" of the project.
2.  **Make Decisions**: Refer to Gamification Theory when designing new mechanics.
3.  **Resolve Disputes**: Refer to the Manifesto or Context when vision is unclear.
